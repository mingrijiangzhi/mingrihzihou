# 题目

L1-065 嫑废话上代码 (5 分)
Linux 之父 Linus Torvalds 的名言是：“Talk is cheap. Show me the code.”（嫑废话，上代码）。本题就请你直接在屏幕上输出这句话。

输入格式：
本题没有输入。

输出格式：
在一行中输出 Talk is cheap. Show me the code.。

输入样例：
无
输出样例：
Talk is cheap. Show me the code.
# 代码
```
#include<stdio.h>
int main()
{
	printf("Talk is cheap. Show me the code.");
	return 0;
}
